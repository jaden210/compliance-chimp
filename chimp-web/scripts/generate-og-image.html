OG Image Generator

This page generates the Open Graph image for social media previews. To create the image:

  1. Open this HTML file in Chrome
  2. Open DevTools (F12 or Cmd+Option+I)
  3. Press Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows) and type "screenshot"
  4. Select "Capture node screenshot"
  5. Click on the OG image container below
  6. Save the image as og-image.png
  7. Copy to src/assets/og-image.png

OR use an online tool like htmlcsstoimage.com with the HTML below.

Required dimensions: 1200 x 630 pixels
